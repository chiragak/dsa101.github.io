Installation

For VS Code:

  • Install Java Development Kit (JDK) if you don't already have it. Download from here
  • Install the Java Extension Pack in VS Code. This will allow VS Code to understand Java code.
  • Open your Java project folder in VS Code. The extension will automatically detect and configure the Java runtime.
  • You may need to set the "java.home" variable in VS Code settings to point to your JDK installation location.

For Eclipse:

  • Install the Eclipse IDE for Java Developers. Download from here
  • Launch Eclipse and create a new Java project. It will automatically detect and configure the installed JDK.
  • If Eclipse does not detect your JDK, go to Window > Preferences > Java > Installed JREs and add the JDK location.
  • You may also need to set the JRE system library in your Eclipse project to point to the installed JDK. Right click on project > Build Path > Configure Build Path
