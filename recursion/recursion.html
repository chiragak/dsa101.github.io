RECURSION

Recursion is a programming technique where a function calls itself directly or indirectly.

It is used to solve problems that can be broken down into simpler subproblems of the same type.

A recursive function has a base case (termination condition) that stops the recursion, and a recursive step that calls the function again to advance towards the base case.

int factorial(int n) {
            if (n == 1) // base case
               return 1; 
          
            return n * factorial(n-1); // recursive call
          }

Recursion follows a top-down approach - divide the problem into subproblems and apply recursion to solve them.

It uses a stack to keep track of function calls. Stack helps unwind and build the solution step-by-step.
