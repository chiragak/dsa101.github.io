DATATYPES IN JAVA

  • int: used for storing integers (whole numbers) with a size of 4 bytes.
  • double: used for storing decimal numbers with a size of 8 bytes.
  • float: used for storing decimal numbers with a size of 4 bytes.
  • boolean: used for storing true/false values with a size of 1 byte.
  • char: used for storing single characters with a size of 2 bytes.
  • String: used for storing sequences of characters with a size that varies depending on the length of the string.

It is important to choose the appropriate data type for your variable, as this can affect the amount of memory used by your program and the accuracy of your calculations. For example, if you are working with decimal values, you should use the double data type instead of the float data type, as the double data type is more precise.

Here is an example of how to declare and initialize variables of different data types:

// int
          int age = 25;
          
          // double
          double price = 19.99;
          
          // float
          float temperature = 98.6f;
          
          // boolean
          boolean isStudent = true;
          
          // char
          char grade = 'A';
          
          // string
          String name = "John Doe";

These variables can be used in expressions just like the variables in the previous example.
