COMMENTS

Comments are used to explain code and make it more readable. They are ignored by the compiler and do not affect the program's functionality.

Comments can be single-line or multi-line:

// This is a single-line comment
    /*
    This is a multi-line comment
    */

Comments can also be used to temporarily disable code:

// int x = 5;

In this example, the line of code that declares and initializes the variable 'x' is commented out and will not be executed.

It is important to use comments to explain complex code or to provide context for other developers who may be working on the same project.
