VARIABLES

What are variables? Variables are like containers that store values that can change. We need to give each variable a descriptive name so we know what kind of value it holds.

For example, let's say we want to create variables to store the dimensions of a rectangle - the length and the width.

We would declare two variables like this:

int length;
          int width;

The int means that these variables will hold integer numbers. Then we give descriptive variable names - length and width.

Before we can calculate the area, we need to store the actual length and width values in these variables. Let's say the length of our rectangle is 5 units, and the width is 3 units.

We would assign values to the variables:

length = 5; 
          width = 3;

Now the length variable contains the number 5, and the width variable contains the number 3.

To calculate area, we need to multiply length * width. So we can create another variable area to store the result:

int area;
          area = length * width;

Using variables allows us to easily calculate the rectangle's area for any length and width!
