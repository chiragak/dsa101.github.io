TYPE CASTING

Type casting is a method or process that converts a data type into another data type.

There are two types of type casting:

  1. Widening Type Casting
  2. Narrowing Type Casting

Widening Type Casting

Widening type casting is when we convert a smaller data type to a larger data type. In this process, we don't lose any information.

For example, converting an int to a float.

int myInt = 9;
float myFloat = myInt;  // Automatic casting: int to float
System.out.println(myInt);   // Outputs 9
System.out.println(myFloat); // Outputs 9.0

Narrowing Type Casting

Narrowing type casting is when we convert a larger data type to a smaller data type. In this process, we might lose some information.

For example, converting a float to an int.

float myFloat = 9.78f;
int myInt = (int) myFloat;  // Manual casting: float to int
System.out.println(myFloat);   // Outputs 9.78
System.out.println(myInt); // Outputs 9
