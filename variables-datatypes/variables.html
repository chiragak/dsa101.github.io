VARIABLES

What are variables?
Variables are like containers that store values that can change. We need to give each variable a descriptive name so we know what kind of value it holds.

For example, let's say we want to create variables to store the dimensions of a rectangle - the l and the b.

We would declare two variables like this:

int l;
int b;

The int means that these variables will hold integer numbers.

Before we can calculate the area, we need to store the actual length and breadth values in these variables. Let's say the length of our rectangle is 2 units, and the breadth is 8 units.

We would assign values to the variables:

l = 2;
b = 8;

Now the variable l contains the number 2, and the variable b contains the number 8.

To calculate area, we need to multiply length * breadth. So we can create another variable area to store the result:

int area;
area = l * b;
System.out.println(area)

Using variables allows us to easily calculate the rectangle's area for any length and breadth!
